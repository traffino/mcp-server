## Shared Mailbox

Alle Mail-Endpoints unterstuetzen Shared Mailbox via `user` Parameter:
- Ohne `user`: `/me/...` (eigenes Postfach)
- Mit `user`: `/users/{userPrincipalName}/...` (delegiertes Postfach)

## Authentifizierung

OAuth 2.0 Client Credentials Flow:
- Token-Endpoint: `https://login.microsoftonline.com/{tenant}/oauth2/v2.0/token`
- Scope: `https://graph.microsoft.com/.default`
- Token wird cached und automatisch erneuert (60s vor Ablauf)
